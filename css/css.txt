Css is used to design our websites.
Css can be used in 3 ways. They are 1) Inline 2)Inside the html file 3) Creating a css file and linking it inside the head of html file.
Css syntax: Selector{
    property: Value;
}

selectors{
    id(#),tag name,class(.),
}

@media queries: max-width: 600px ( Changes will be made for devices <=600px)
@media queries: min-width: 600px ( Changes will be made for devices >=600px)
RWD - always design for the smaller devices first and use media queries to make the change for devices with larger screen.
